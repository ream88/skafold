    • Skafold

    HTML scaffolding

    Project maintained by haihappen Hosted on GitHub Pages — Theme by mattgraham

    Generates HTML scaffolds the way I need it everyday as a web developer.

    The days were I needed only a index.html as scaffold for web apps are long gone. Today my work depends on various different frameworks:

    • CoffeeScript
    • jQuery
    • Backbone.js (not implemented yet)
    • Sass
    • Foundation (not implemented yet)
    • Bourbon (not implemented yet)
    • Sprockets

    Usage

    Skafold simplifies the generation of such a scafold by reducing it into one command (using the incredible thor by @wycats):

    skafold APP_NAME

    Installation

    gem install skafold
